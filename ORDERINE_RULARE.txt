ORDINEA DE RULARE A PROIECTULUI (PAS CU PAS)
=================================================

Acest fisier descrie clar ordinea in care se ruleaza componentele proiectului,
cu scopul fiecarui pas. Sunt doua fluxuri principale:
1) Pregatirea datelor + antrenare (inainte de a folosi site-ul)
2) Fluxul de predictie cand apesi Predict in UI

-------------------------------------------------
1) ORDINEA INAINTE DE SITE (PREGATIRE + ANTRENARE)
-------------------------------------------------
Scop: sa generezi datele, sa le pregatesti, sa antrenezi modelul si sa obtii
fisierele necesare pentru predictie (model + scaler + split-uri).

PAS 1: Generare date sintetice (chair)
- Script: src/data_acquisition/generate_chairs.py
- Ce face: creeaza dataset-ul sintetic cu parametrii geometrici.
- Output: data/generated/chairs_dataset.csv
- De ce e necesar: acesta este setul de date de baza (nu exista date reale).

PAS 2: Curatare + validare
- Script: src/preprocessing/chair_data_cleaner.py
- Ce face: verifica lipsa valorilor, reguli logice (ex: spatar doar cand has_backrest=1).
- Output: data/processed/chairs_clean.csv
- De ce e necesar: elimina combinatiile invalide si confirma consistenta datelor.

PAS 3: Scalare
- Script: src/preprocessing/chair_feature_scaler.py
- Ce face: aplica StandardScaler pe toate caracteristicile.
- Output: data/processed/chairs_scaled.csv + config/chair_scaler.pkl
- De ce e necesar: RN-ul invata mai bine cand valorile sunt in aceeasi scara.

PAS 4: Split train/validation/test
- Script: src/preprocessing/chair_data_splitter.py
- Ce face: imparte datele in 70% train, 15% validation, 15% test.
- Output:
  - data/chairs/train/X_train.csv, y_train.csv
  - data/chairs/validation/X_val.csv, y_val.csv
  - data/chairs/test/X_test.csv, y_test.csv
- De ce e necesar: separa corect datele pentru antrenare si evaluare.

PAS 5: Antrenare model (chair)
- Script: src/neural_network/train_chair.py
- Ce face: incarca datele din train/validation, construieste arhitectura din model.py,
  antreneaza RN si salveaza modelul.
- Output:
  - models/chair_model.h5
  - results/chair_training_history.csv
- De ce e necesar: acesta este modelul care va fi folosit la predictie in UI.

PAS 6: Evaluare (optional, dar recomandat)
- Script: src/neural_network/evaluate.py
- Ce face: testeaza modelul pe data/chairs/test.
- Output:
  - results/chair_test_metrics.json
- De ce e necesar: confirma performanta modelului si verifica ca tot fluxul functioneaza.

PAS 7: Export ONNX + benchmark (optional, pentru Nivel 3)
- Script: src/neural_network/export_onnx.py
- Ce face: exporta modelul in ONNX si masoara latenta (CPU, batch=1).
- Output:
  - models/chair_model.onnx
- De ce e necesar: demonstratie compatibilitate ONNX + latenta sub 50ms.

-------------------------------------------------
2) ORDINEA CAND APESEZI "PREDICT" IN SITE
-------------------------------------------------
Scop: sa folosesti modelul antrenat pentru a prezice clasa obiectului si sa
obtii scriptul Blender + preview.

PAS 1: UI primeste inputul utilizatorului
- Fisier: src/app/main.py
- Ce se intampla: utilizatorul completeaza parametrii si apasa Predict.

PAS 2: Incarcare model + scaler (daca nu sunt in cache)
- Fisier: src/app/main.py
- Se incarca:
  - models/chair_model.h5 (model antrenat)
  - config/chair_scaler.pkl (scaler-ul folosit la preprocesare)
- Scop: aceleasi transformari ca la antrenare.

PAS 3: Preprocesare input
- Fisier: src/app/main.py
- Ce se intampla: inputul este pus intr-un DataFrame si scalat.

PAS 4: Inferenta RN
- Fisier: src/neural_network/model.py (arhitectura) + model incarcat
- Ce se intampla: modelul produce probabilitati pe clase si clasa prezisa.

PAS 5: Generare script Blender
- Fisier: src/blender_scripts/chair_generator.py
- Ce se intampla: pe baza clasei prezise se genereaza scriptul pentru model 3D.

PAS 6: Render preview (daca Blender API ruleaza)
- Fisier: src/blender_api/server.py
- Ce se intampla: scriptul este trimis catre Blender API pentru render.

PAS 7: Afisare rezultat
- Fisier: src/app/main.py
- Se afiseaza:
  - clasa prezisa
  - probabilitati
  - script Blender
  - preview (daca Blender API este disponibil)

-------------------------------------------------
REZUMAT (fisiere cheie)
-------------------------------------------------
- Arhitectura RN: src/neural_network/model.py
- Antrenare chair: src/neural_network/train_chair.py
- Evaluare chair: src/neural_network/evaluate.py
- Export ONNX: src/neural_network/export_onnx.py
- UI: src/app/main.py
- Blender generator: src/blender_scripts/chair_generator.py
- Blender API: src/blender_api/server.py
